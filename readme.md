# Python e Inteligência Artificial no Contexto da Engenharia de Software Aula 1

Este repositório contém o código-fonte de apoio para o livro "Python e Inteligência Artificial no Contexto da Engenharia de Software", de Edson Orivaldo Lessa Junior.

## Sobre o Projeto

Este projeto oferece exemplos práticos e material de apoio para o livro, focando na aplicação de Python e Inteligência Artificial na Engenharia de Software.

## Começando

Esta seção guiará você na configuração do ambiente de desenvolvimento para executar os exemplos do livro.

### Pré-requisitos

* Python 3.x
* pip (gerenciador de pacotes do Python)

### Instalação

1. Clone este repositório: `git clone <URL_DO_REPOSITORIO>`
2. Navegue até o diretório do projeto: `cd unicesumar-py-ia-eng-sw`
3. Instale as dependências: `pip install -r requirements.txt` (Será necessário criar o arquivo `requirements.txt`)

## Estrutura do Projeto

* **data/**: Contém os datasets utilizados nos exemplos.
* **ex01_numpy_basico/**: Exemplos introdutórios sobre a biblioteca NumPy.
* **ex02_pandas_dataframe/**: Exemplos sobre manipulação de dados com Pandas.
* **ex03_eda_csv/**: Análise exploratória de dados a partir de arquivos CSV.
* **ex04_tratamento_nulos/**: Técnicas para lidar com dados ausentes.
* **ex05_visualizacao_matplotlib/**: Exemplos de visualização de dados com Matplotlib.
* **ex06_visualizacao_seaborn/**: Exemplos de visualização de dados com Seaborn.
* **main.py**: Arquivo principal para execução de exemplos específicos.
* **check_version.py**: Script para verificar as versões das bibliotecas instaladas.

## Como Usar

Execute os scripts `.py` individualmente para ver os exemplos em ação.
